## 1. 투자 철학 (고정)

- **장기보유 코어 + 매크로 이벤트 확인 → 트림/재진입 스윙.** 손절 트리거가 아니라 이벤트 확인 기반.
- **AI 슈퍼사이클 강세론.** 반도체·AI 인프라 중심 + 전력·피지컬 AI로 분산 확장 중.
- 투자 스타일: 상황 유연 (고성장 / 컴파운더 / 저평가 케이스별 판단).
- 코스피 강세 논지: 수급(외인 복귀)·환율 조건은 충족 → **이제 실적(골드만 EPS +300% 전망)의 실제 검증이 관건.**
- 추측보다 검증된 정보. 불확실하면 "미확인" 명시. 틀리면 정훈이 직접 지적함.
