## 1. 투자 철학 (고정)

- **장기보유 코어 + 매크로 이벤트 확인 → 트림/재진입 스윙.** 손절 트리거가 아니라 이벤트 확인 기반.
- **AI 슈퍼사이클 강세론.** 반도체·AI 인프라 중심 + 전력·피지컬 AI로 분산 확장 중.
- 투자 스타일: 상황 유연 (고성장 / 컴파운더 / 저평가 케이스별 판단).
- 코스피 강세 논지: 수급(외인 복귀)·환율 조건은 충족 → **이제 실적(골드만 EPS +300% 전망)의 실제 검증이 관건.**
- **🔑 외국인 수급 = 코스피 메인 엔진 (핵심 추적 신호) [6/14 격상]**: 외인 순매수 복귀(6/12 +2.1조, 25거래일 만 전환)는 **상승 추세 신호**이자 반도체 투톱(삼성·SK하이닉스) 매수 = HBM/AI 메모리 순풍. **단 첫 전환은 초기 신호 — 지속성(연속 일수·누적액)이 관건.** **외인이 순매도로 재전환하면 강세 논지 약화 = 신중 신호.** 외인 수급 연속성을 **SK하이닉스 등 반도체 진입 타이밍의 스위치**로 사용(외인 매수 지속=선취 명분↑ / 식으면=느긋하게).
- 추측보다 검증된 정보. 불확실하면 "미확인" 명시. 틀리면 정훈이 직접 지적함.
- **🔑 데이터 기반 우선 원칙 [6/15 정훈 강조]**: 미래 시장 가능성 베팅도 좋지만 **실적 데이터 기반이 없으면 결국 되돌림**(예: 두산로보틱스 적자·PSR 219배). 코어 비중은 데이터로 검증된 종목에. **단기 긍정 이슈로 오르는 모멘텀주를 먹고 파는 것도 OK — 단 코어가 아닌 트레이딩 대상으로 분류하고, 매일 분석·잘 빠질 것.** 종목별 [코어=데이터 / 모멘텀=테마·잘빠지기] 구분을 항상 명시.
