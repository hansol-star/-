# 콜 스코어카드 — 캘리브레이션 누적 원장

> self-review 스킬(주간 첫 보고서·R4 토요일 루틴)이 **이 파일 맨 위에 이번 주 스코어카드를 prepend**한다(최신이 위).
> 기계 원장 = `data/app/calls_log.jsonl`(score_calls.py가 append). 이 .md = 사람이 읽는 캘리브레이션 추세.
> 투자 자문 아님 — 후행검증·분석 참고.

## 형식 (매주 prepend, 아래 골격 복제)

```
### 주간 스코어카드 — {YYYY-MM-DD} (표본 {N}콜 · 기간 {시작}~{끝})
- **별점 캘리브레이션**: ⭐4~5 평균 전진 {x}% vs ⭐1~2 {y}% (순서 정상/역전?)
- **방향 적중률**: {맞은 콜}/{전체} = {%}
- **매수존 진입률**: 제시 매수존에 실제 도달 {%}
- **목표가 터치율**: 목표 방향 도달 {%}
- **Brier 점수**: {score} (낮을수록 좋음 · 0.25=무작위 · 과신 플래그 여부)
- **경제사냥꾼 [정정] 비율**: hunter_score.py 누적 정정률 {%}/검증률 {%}
- **편향 플래그**: (별점 쏠림·순서 역전·반복 빗나감 — 발견 시)
- **교정 제안(정훈 승인 대기, 자동변경 ❌)**: {채점기준/전망 조정안 or 없음}
- **표본 주의**: 결과론 함정 경계 — 단기 노이즈로 과벌 금지, 기간·표본 명시.
```
